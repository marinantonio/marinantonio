SPLIT BUS

>10.000 downloads
>2.000 active users

First Split Bus version appeared on Play Store back in september 2013. It was my first Android app with simple idea, bring public transport closer to citizens. The local transport company, Promet Split had simple Joomla website with a timetables displayed as pictures. Back then, a timetables didn't change often, maybe twice per year so first Split Bus version had all pictures saved as assets. When the timetable change occurred, I would release an update. In following months, as my coding skills improved, first I've used WebView and later built an entire new logic for saving pictures to internal sdcard which can be used for later offline viewing.

    Features:
  • Shows all urban area and suburban area buses and their timetables
  • Shows city maps containing bus stations and bus routes
  • Shows the latest news, road works and other changes in bus routes
  • Shows various useful informations (tariff zones, sms parking numbers etc.)
  • Using GPS app can find nearby bus stations
  • Once you download timetable you wont need internet access anymore
  • Simple and fast design
  • Optimized for tablets
  • Completely free, without any ads
  • Supports Android 4.0 and newer

From the release app was free and without any ads. Nobody responsible from the company ever contacted me, few times in interviews they said they were working on mobile solutions but none of that ever happened. This year I started to rewrite entire Java code to Kotlin. I've always wanted to open source Split Bus, but I never had time to tidy up code a little bit. Kotlin provided a perfect excuse, so full development is once again in progress.
